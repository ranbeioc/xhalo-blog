This is the default welcome test article for a new xHalo Blog site.

Use this starter post to verify that Hexo, the NexT theme, Cloudflare Pages headers, article permalinks, and post asset folders are all working before importing historical content.

When you are ready to migrate an existing Hexo blog, run:

```bash
npm run init:hexo-next -- --target ../my-blog-test --source ../hexo-blog --site-url https://my-blog-test.pages.dev
```

The importer copies only safe Hexo/NexT content and disables deploy targets by default.
