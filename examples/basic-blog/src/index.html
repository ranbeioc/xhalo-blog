xhalo-blog / examples/basic-blog

Minimal static baseline

This example is the smallest deployable public site in the repository. It is intentionally separate from the Hexo plus NexT example and exists to verify a plain Cloudflare Pages path.

What this example keeps:

  • static HTML only
  • Cloudflare Pages _headers support
  • no production content, analytics IDs, or comment endpoints

Suggested Pages settings:

  • Root directory: examples/basic-blog
  • Build command: npm run build
  • Build output directory: dist
